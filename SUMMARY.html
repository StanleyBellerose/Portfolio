💡 Conseils importants

Format vidéo recommandé

✓ MP4 — Format universel, compatible partout

✓ WebM — Format léger, optimisé pour le web

Utiliser les deux pour compatibilité maximale

Paramètres d'optimisation

  • Résolution: 1920×1080 maximum
  • Bitrate: 1000-3000 kbps
  • Durée: 15-60 secondes (web)
  • Taille fichier: < 10-20 MB
