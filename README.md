### 💡 Conseils

- **Format**: MP4 (universel) + WebM (optimisé web)
- **Taille**: 1920x1080 max, 1-3 Mbps
- **Durée**: 15-60 secondes pour web
- **Compression**: Utilisez [HandBrake](https://handbrake.fr/) ou [FFmpeg](https://ffmpeg.org/)
